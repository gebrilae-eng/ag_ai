# /speckit.constitution

Create `.ai/spec/memory/constitution.md` using `.ai/spec/templates/constitution-template.md`.

1. Load existing constitution if present
2. Collect/derive values from `$ARGUMENTS` and project context
3. Define 3-5 principles: clear name, rules (MUST/MUST NOT), rationale
4. Add project-specific constraints
5. Write constitution file
6. Report: version, principles defined, next step

## Arguments
$ARGUMENTS
